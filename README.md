# Алабуга Политех — главная страница (прототип)

React/Vite прототип главной страницы платформы по вайрфрейму, на дизайн-системе `borrom-ds-test` и токенах из `design-system-template`.

## Запуск

```bash
cd platform
npm install
npm run dev
```

Откройте http://127.0.0.1:5173

## Реализованные механики

- **Шапка** — навигация с активным пунктом, уведомления, аватар
- **Сторис** — горизонтальная прокрутка
- **Направления** — карусель + фильтры (Chips)
- **Мероприятия** — карусель со стрелками, теги, hover на карточках
- **Новости** — крупная карточка + список, фильтры
- **Стрим-баннер** — обратный отсчёт (таймер обновляется каждую секунду)
- **Статьи** — сетка 3 колонки + фильтры по категориям

Тема по умолчанию: **тёмная** (как на вайрфрейме).

## Структура

- `src/data/homeContent.ts` — мок-данные
- `src/components/home/` — секции страницы
- `src/components/ui/` — карусель, заголовок секции
- `src/hooks/useCountdown.ts` — таймер для эфира
